Interactive Responsive Website

A responsive front-end website built using HTML, CSS, JavaScript, and Bootstrap.

Features: 
- Responsive layout for desktop and mobile
- Interactive hero cards with expandable information sections
- Image carousel for featured content
- Styled UI components and navigation

Technologies Used: 
- HTML
- CSS
- JavaScript
- Bootstrap

Note: 
Interactive items redirect to existing external websites related to the featured items. These are included to simulate real navigation within a production site.
